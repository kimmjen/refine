# Code of Conduct

이 프로젝트는 **Contributor Covenant v2.1** 을 행동 규범으로 채택합니다.

전문 (한국어 번역 포함):
- English: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- 한국어: <https://www.contributor-covenant.org/ko/version/2/1/code_of_conduct/>

## 신고 채널

위반 사례는 보안 신고와 동일한 채널로:

- [GitHub Security Advisory](https://github.com/kimmjen/refine/security/advisories/new) (비공개)
- 또는 메인테이너에게 직접 연락

영업일 기준 7일 내 1차 응답을 목표로 합니다.

## 적용 범위

- 본 repo 의 issue / PR / discussion
- 메인테이너에게 보내는 직접 연락 (이메일 / DM)

문의 / 명확화 필요 시 issue 로 자유롭게 질문해주세요.
